# 🏔️ 趣享攀登 — 户外运动场馆全业务管理平台


面向攀岩运动场馆的 SaaS 化管理平台，覆盖用户端小程序（C端）、教练端子系统、管理后台（B端）三大终端。


全程使用 **Cursor + Claude** AI 辅助开发，单人 5 周内完成从设计到部署的全链路交付。


## 📱 功能概览


### 用户端（微信小程序 · 47 页面）
- 场馆首页（轮播图 / 内容管理）
- 课程预约（团课 / 私教，日期筛选 + 分类导航）
- 商城系统（分类侧边栏 + 购物车 + 订单 + 微信支付）
- 个人中心（训练统计 / 积分 / 保险状态 / 二维码核销）
- 会员卡体系（次卡 / 期限卡，购买 → 激活 → 核销 → 过期）


### 教练端
- 课程管理（团课 / 私教 CRUD）
- 预约审核（批量操作）
- 配送单管理 / 扫码核销


### 管理后台（12 模块）
- 活动 / 课程 / 用户 / 商品 / 会员卡管理
- 多类型审核（报名 / 保险 / 退款）
- 场馆首页内容编辑 / Excel 导出
- 操作日志 / 错误日志 / 数据备份


## 🛠️ 技术栈


| 层级 | 技术 |
|------|------|
| **前端 C 端** | 微信小程序原生（分包架构） |
| **后端** | Python + Flask，90+ RESTful API |
| **数据库** | MySQL（24 张业务表） |
| **支付** | 微信支付 V3（JSAPI / RSA 签名 / AES-GCM 解密 / 退款） |
| **部署** | Docker Compose + Nginx + GitHub Actions CI/CD |
| **AI 工具** | Cursor / Claude |


## ⚙️ 核心技术亮点


- **RBAC 三级权限**：装饰器模式实现用户 / 教练 / 管理员权限控制
- **微信支付 V3 全流程**：统一下单、RSA-SHA256 签名、AES-256-GCM 回调解密、退款
- **DBUtils 连接池** + 内存令牌桶限流 + 三级日志审计（操作 / 错误 / 审计）
- **HMAC-SHA256 签名限时二维码核销系统**
- **行级锁（SELECT FOR UPDATE）**保障并发场景下的数据一致性
- **积分体系** + 会员卡全生命周期管理（购买 → 激活 → 核销 → 过期自动检测）
